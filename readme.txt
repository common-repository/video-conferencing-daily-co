Video conferencing with Daily.co API

== Description ==

Video conferencing with Daily.co offers Screen Sharing, Chat mode, Audio/Video conference with two or more users.

By using this plugin, you can create rooms from within your WordPress dashboard and manage them just like you do on Daily.co dashboard.

Created rooms, will provide you a quick shortcode snippet which you can place it to any of your posts or pages. Whether it's only visible after a membership with content restriction or after a completion of a form or course completion.

**Shortcode**

`
[dailyco_meeting name="hello-world" width="100%" height="400px" show_leave_btn="true" show_fullscreen_btn="true"]
`

**Options**

1. `name` [Required]
2. `width` [Optional] Default `100%`
3. `height` - [Optional] Default `400px`
4. `show_leave_btn` - [Optional] Default `true`
5. `show_fullscreen_btn` - [Optional] Default `true`

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
